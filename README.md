# Darwin

Darwin is an independent Progressive Web App (PWA) designed to help Brazilian real estate buyers estimate **construction phase financing charges** ("juros de obra") during off-plan property financing.

The app was created to improve transparency around monthly financing costs, allowing users to simulate, track, and project installment values throughout the construction period.

## Features

* Guided onboarding flow for financing setup
* Monthly installment estimation
* Construction progress tracking (% completed)
* Editable payment timeline
* Profile saving with local storage
* Financial summaries and projections
* Freemium model (basic insights free, full projections premium)

## Important Disclaimer

Darwin is an independent project and is **not affiliated with banks, financial institutions, developers, or construction companies**.

Calculations are based on the official construction financing formula commonly used in Brazil, but all values generated are **estimates only** and do not replace official charges communicated by the financing institution.
